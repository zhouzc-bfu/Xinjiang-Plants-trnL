# Xinjiang Plants trnL P6-loop Reference Database

## Overview
This repository contains a custom trnL P6-loop reference database of desert plants from Xinjiang, China. All relevant information is included in the FASTA headers. 

**Note:** This database was created using [OBITools](https://pythonhosted.org/OBITools/), a software suite for DNA metabarcoding analyses.

## Contents
- `data/Xinjiang_trnL.fasta`: FASTA file with full sequence information in headers, generated via OBITools
